Install C Compiler & Run Hello World

Step 1: Install C Compiler (GCC)

Windows: Download and install MinGW or use Code::Blocks with GCC.

Linux: Run sudo apt install build-essential

Mac: Install Xcode Command Line Tools using xcode-select --install

Step 2: Configure IDE

Install an IDE like VS Code, Code::Blocks, or Dev-C++.

Set compiler path (GCC) in IDE settings.

Step 3: Write Hello World Program

#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}

Step 4: Compile & Run

gcc hello.c -o hello
./hello

Output:

Hello, World!
